Для коректного відображення кирилиці в PDF-експортах необхідно покласти
сюди файл шрифту з підтримкою Unicode під назвою:

    DejaVuSans.ttf

Завантажити можна з:
    https://dejavu-fonts.github.io/Download.html

Без цього файлу PDF буде згенеровано стандартним шрифтом Helvetica,
який НЕ підтримує кирилицю (символи будуть відображатись як квадрати).

CSV-експорт працює коректно без додаткових налаштувань.
